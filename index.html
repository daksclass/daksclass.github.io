  • Just printing stuff is boring. Let's make this interactive!
  • First, import Scanner. Scanner is a tool we use to process input from a user into something that our java program knows how to use.
  • After you import Scanner, you need to create an instance. Importing scanner is basically like accessing a factory of tools, creating an instance is using that factory to make your very own scanner that you can use!
  • Creating an instance isn't too bad, it's a lot like creating a new variable. First tell it it's type. It's type is Scanner. Then name it. I usually stick with something clear, like input. Then you put in some code that just sells it that's it's a new Scanner, looking for code going into the program from the user. Tada!
  • Once we have a Scanner, we can ask a question and then use out Scanner, which we named input, to process the answer and save it in a variable.
  • The Scanner needs to know what data type it's looking for. Is it an int? A string?
import java.util.Scanner; public class ScanStuff{ public static void main(String[] args){ Scanner input = new Scanner(System.in); System.out.print("What... is your name?"); String answerName = input.nextLine(); System.out.print("What... is your quest?"); String answerQuest = input.nextLine(); System.out.print("What... is the airspeed velocity of an unladen swallow?"); int answerVelocity= input.nextInt(); } }
