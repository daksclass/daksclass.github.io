Proclipsing

Here's how to install Proclipsing, aka Processing with Eclipse. The code lives here: https://github.com/ybakos/proclipsing.

  1. Just above Github's file list, there are several tabs. Go to Releases.
  2. Go to current_releases and download the archive type of your choice (zip or tar.gz)
  3. Extract the archive someplace easy to get to.
  4. In Eclipse, go to Help > Install Additional Software, and click the Add button.
  5. Enter a name (probably "Proclipsing" but it's not super-important), then click the Local button next to the name field.
  6. Find the folder you extracted the archive to and select it. Proceed with installation.

Processing

Here's how to install Processing outside of Eclipse. We didn't use it a ton, but it's an easy option if Proclipsing is too much of a pain. https://processing.org/download/.

  1. Just download Processing. It's super easy.
  2. Copy and paste your code into a new Processing sketch. Take out imports and the outermost class that Eclipse auto adds. Keep all variables, setup(), draw() and any other functions you wrote.
  3. If you have added images or music, import them into your data folder in Processing.
