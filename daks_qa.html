Q:How do other languages compare to Java?

A:I'm gonna break this down.

  1. Python - has a lot less syntax stuff. Here's what printing looks like in Python: print ("hello world"). Pretty simple, huh? It's generally a bit slower of a programming language. Python is used as a glue language, which people sue to connect a bunch of programming components, while Java is more bottom up.
  2. C++ - much faster than Java, but C++ code can end up taking up more memory. Here's what printing looks like in C++: std::cout << "Hello World!";
  3. HTML - not even remotely similar. Used for website content and structure.
  4. Javascript - Java is to JavaScript as ham is to a hamster. Java is a general purpose programming language; back in the day, JavaScript was used on websites to make them interactive. It's used for more than that now, but it's still very different.
